# 🚀 SkillNova - Course Platform

SkillNova is a modern course platform built using React. Users can explore courses, view details, and experience a clean UI with light/dark mode.

---

## ✨ Features

- 📚 Course listing
- 🔍 Search functionality
- 📄 Course details page
- 🌙 Light / Dark mode
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React.js
- CSS3
- React Router
